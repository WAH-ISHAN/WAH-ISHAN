## Featured Projects

### Create-BackList – Backend Generator CLI Tool (2025)

Published command-line tool (NPM package) that automatically generates backend servers by analyzing frontend code.

- Scans frontend projects to detect API calls and auto-generates <strong>routes</strong>, <strong>controllers</strong>, and <strong>models</strong>
- Supports backend generation in <strong>Node.js</strong>, <strong>Java</strong>, <strong>C#</strong>, and <strong>Python</strong>
- Includes built-in <strong>JWT authentication</strong> and sample data seeding
- Docker-ready setup for fast deployment
- Helped reduce backend setup time by ~<strong>50%</strong>

<strong>Tech:</strong> Node.js, TypeScript, EJS, Babel, Inquirer.js

---

### Learning Management System (LMS) – Full-Stack App (2025)

Role-based LMS platform with access levels for <strong>Admin</strong>, <strong>Lecturer</strong>, and <strong>Student</strong>.

- Implemented secure <strong>JWT-based authentication</strong> and authorization with <strong>Spring Security</strong>
- Built RESTful <strong>Spring Boot</strong> backend with JPA for users, courses, enrollments, content, and calendars
- Developed <strong>React</strong> frontend for login/registration, course listing, enrollment, content viewing, and per-course calendar
- Applied best practices in <strong>API design</strong>, input validation, and role-based access control
- Used <strong>MySQL</strong> for persistent storage

<strong>Tech:</strong> React.js, Spring Boot, Java, JWT, MySQL, JPA/Hibernate

---

- Worked on user management modules and API communication workflows

<strong>Tech:</strong> React.js, Java, REST APIs, MySQL Workbench
